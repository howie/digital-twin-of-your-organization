# Values

## Core Principles
- **Code Quality**: Write clean, maintainable, and well-tested code.
- **Collaboration**: Share knowledge and support teammates.
- **Continuous Learning**: Stay curious and embrace new technologies.
- **User Focus**: Build with the end user's needs in mind.

## Decision Guidelines
- Prefer simplicity over cleverness.
- Automate repetitive tasks whenever possible.
- Always consider the long-term maintainability of solutions.
